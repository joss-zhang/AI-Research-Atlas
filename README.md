# AI-Research-Atlas 🧠🗺️

> AI 研究知识图谱 —— 论文精读、概念拆解与实验手稿。

在人工智能领域长期研究与学习的核心沉淀。与普通代码仓库不同，这里**以笔记和资料为主，代码仅为验证辅助**。

## 🗂️ 仓库导览

| 目录 | 内容说明 |
| :--- | :--- |
| **`01-papers/`** | 顶会论文精读笔记，包含核心贡献、方法拆解、优缺点分析 |
| **`02-concepts/`** | 核心概念深度重述，用“自己的话”讲清楚底层原理 |
| **`03-code/`** | 轻量实验代码（如 Toy Example），验证笔记中的关键推导 |
| **`04-resources/`** | 外部优质课程、博客、数据集的索引与评价 |
| **`05-journals/`** | 阶段性学习日志，记录思考过程与方向调整 |

## 🧭 使用原则

1. **重质量轻数量**：每一篇笔记必须是“消化后”的输出，而非原文搬运。
2. **双向链接**：鼓励在不同概念文件间互相引用，构建知识网络。
3. **代码极简**：`03-code/` 中的脚本需控制在百行以内，专注于单个原理验证。
